# Sexo

## Configuration
The bot can be easily configured by creating the file `sexo.yml` in the same working directory Sexo is ran from.

Here is an example configuration:

```yaml
dbUrl: jdbc:sqlite:sexo.db
nick: sexo
login: sexo
realName: Sexo
server: irc.efnet.org
port: 6667
channels:
  - "#camp truetocaesar"
```
